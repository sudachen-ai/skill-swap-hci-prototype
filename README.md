# Skill Swap Mobile App Prototype

## Overview

Skill Swap is a mobile app prototype designed to help busy adults learn practical skills through short peer-to-peer micro-learning sessions.

The platform supports matching, scheduling, live micro-sessions, progress sharing, and community interaction.

## My Role

I worked on Scenario 2: Micro Session flow, including:

- Waiting room
- Live session interface
- Floating video window
- Post-session progress feedback
- Manage micro-session page

## Tools

## Key Contributions

- Designed the Micro Session user flow for a peer-to-peer learning mobile app.
- Created interactive Figma screens for live sessions and floating video navigation.
- Applied HCI principles including affordance, mental models, cognitive load reduction, and Nielsen usability heuristics.
- Incorporated peer evaluation feedback to improve user control, navigation clarity, and session continuity.
